# Steven's Job Autocompleter

**Steven's Job Autocompleter** is a Chrome Extension designed to automate the repetitive parts of job applications on platforms like LinkedIn and Greenhouse. It combines traditional form-filling logic with Google's **Gemini 1.5 Flash AI** to generate personalized, professional answers to open-ended application questions.

## 🚀 Features

* **Intelligent Form Filling**: Automatically detects and fills common fields like Name, Email, Phone, LinkedIn, and GitHub.
* **AI-Powered Responses**: Uses Gemini 1.5 Flash to generate professional 2-sentence answers for "Why are you interested in this role?" based on your background and the specific job description.
* **Smart Radio Buttons**: Handles complex application questions, such as visa sponsorship or demographic info, by identifying keywords and selecting the appropriate options.
* **React-Friendly Input**: Utilizes a custom `setNativeValue` function to ensure automated inputs are correctly recognized by modern web frameworks like React.
* **Secure API Storage**: Provides a simple UI to save your Gemini API key locally in Chrome's storage for easy configuration.
* **Python Integration**: Includes a `spa.py` script leveraging **Playwright** for desktop-level automation and handling "Easy Apply" workflows.

## 🛠️ Technical Stack

* **Extension Framework**: Manifest V3
* **AI Backend**: Google Gemini 1.5 Flash API
* **Frontend**: JavaScript, HTML, CSS (Tailwind styles)
* **Bundling**: Webpack with `dotenv-webpack` for environment variable management
* **Automation**: Playwright (Python)

## 📦 Installation

1.  **Clone the Repository**:
    ```bash
    git clone [https://github.com/Steven-Ou/automated_system.git](https://github.com/Steven-Ou/automated_system.git)
    cd automated_system
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Build the Project**:
    ```bash
    npx webpack
    ```

4.  **Load in Chrome**:
    * Open Chrome and navigate to `chrome://extensions/`.
    * Enable **Developer mode**.
    * Click **Load unpacked** and select the `dist` folder generated by Webpack.

## ⚙️ Configuration

1.  Click the extension icon in your Chrome toolbar to open the popup.
2.  Paste your **Gemini API Key** into the configuration field and click **Save Key**.
3.  The extension is pre-seeded with profile data for **Steven Ou**. To customize the information used to fill forms, update the `profile` object in `popup.js`.

## 📂 Project Structure

* `manifest.json`: Defines extension metadata and permissions.
* `background.js`: Manages communication with the Gemini API.
* `content.js`: Core logic for scanning and filling forms on the webpage.
* `popup.js` / `popup.html`: The user interface for configuration and manual triggers.
* `spa.py`: Playwright script for automated job processing and browser session management.
* `webpack.config.js`: Configuration for bundling the extension source files.
